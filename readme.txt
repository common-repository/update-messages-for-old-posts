Don't let time to make your content outdated

== Description ==
The plugin can create the rules which can be used if - 
1. You have some series of posts about something and you want to put final result link to all posts of series.
2. You have advertised or reported some campaign(contest, exam, job vacancy etc.) which deadline passed.
3. You have some old content which is not actual already. (and it is difficult to edit all of them one by one) 
4. For any other kind of posts which need update messages/ps.-es/comments ...

== Installation ==

1. Upload 'updatepusher.zip' to the '/wp-content/plugins/' directory
2. Unzip it.
3. Go to Dashboard/Plugins and Activate the plugin. 
4. Go to “Dashboard/Settings/Push Update Boxes To Old Content”  for configuring the plugin.
